\chapter{Глава 3. Программная реализация классификатора МРТ снимков }
\label{cha:design}

\section{Программная реализация методики классификации МРТ снимков}

Для реализации методики классификации МРТ снимков был разработан программный комплекс, архитектура которого базируется на математической модели, описанной в предыдущей главе. Комплекс состоит из нескольких программных модулей:
\begin{itemize}
\item Модуль подготовки данных;
\item Модуль обучения;
\item Модуль классификации;
\end{itemize}
